# [!DNL PrestaShop] verbinden

![PrestaShop-Logo](../../../assets/Prestashop-logo.png)

Möchten Sie sehen, wie Ihr Unternehmen funktioniert oder Ihre wertvollsten Akquisequellen und -kanäle finden? Sie können diese Fragen und mehr beantworten, indem Sie Ihre [!DNL PrestaShop] mit [!DNL Commerce Intelligence] verbinden.

Die [!DNL PrestaShop]-Datenbank basiert auf [!DNL MySQL]. Sie können sie über eine [Direktverbindung](../integrations/mysql-via-a-direct-connection.md), [[!DNL cPanel]](../integrations/mysql-via-cpanel.md) oder [über eine `SSH tunnel`](../integrations/mysql-via-ssh-tunnel.md) mit [!DNL Commerce Intelligence] verbinden. Adobe empfiehlt die Verwendung der `SSH`-Methode, da sie die sicherste ist. Informationen zur Überprüfung des SSH-Hostschlüssels finden Sie unter [SSH-Hostschlüsselüberprüfung](../integrations/ssh-host-key-verification.md).
